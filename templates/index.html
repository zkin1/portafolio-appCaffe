CaffeeManagement

Descripción General

CaffeeManagement es una aplicación móvil diseñada para dispositivos Android que simplifica la gestión integral de una cafetería, proporcionando herramientas eficientes tanto para empleados como para administradores.

Funcionalidades Principales

Gestión de Empleados

Registro, aprobación y administración del personal de manera sencilla y eficiente.

Gestión de Productos

Sistema CRUD completo para manejo de menú con interfaz intuitiva.

Gestión de Ventas

Seguimiento y control detallado de las transacciones realizadas.

Características Detalladas

Para Empleados

  • Registro y Acceso
    • Formulario de registro inicial
    • Aprobación del administrador para iniciar sesión
  • Perfil de Usuario
    • Foto de perfil personalizable (opcional)
    • Información personal y contacto
  • Sistema de Comandas
    • Secciones: Pendientes, En Preparación, Entregadas
    • Función de Cancelar Comandas
    • Seguimiento detallado de órdenes

Para Administradores

  • Gestión Avanzada
    • Control total de empleados y menú
    • Generación de reportes y comunicaciones
    • Funcion para generar reuniones
  • Dashboard Administrativo
    • Estadísticas globales de ventas
    • Monitoreo de usuarios activos

Reportes Semanales

  • Ventas por empleado
  • Productos más vendidos
  • Análisis de ventas canceladas
